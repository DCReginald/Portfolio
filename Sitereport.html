Introduction to Web development


Web development is one of the core modules of my software enginnering degree. At the start of the module i was introduced to a platform called Visual Studio Code, which was going to be the main learning platform for the first half of the module. In the first few lessons we began with an introduction to HTML, more specifically the HTML5 language and its syntax, which is used to input the information that needs to be displayed on the website. This was then followed by the introduction to CSS3, a language used to style the website, which was seemingly complicated at first as it was more complex than HTML, but after a few lessons i was able to do quite confortably. The next lessons we were taught how to use the CSS to style the websites, making a menu bar, search bars, textboxes and animations. Which we then used to make our portfolio website, as module assignment.
